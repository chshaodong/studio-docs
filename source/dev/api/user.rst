登录
======================

说明
-----------------------------------------------------------------------------------------------------------------------
- 使用studio需要先登录


**请求方式：**   POST

**请求地址：**   /api/login/


**Content-Type：**
::

    application/json —— 表示传入数据为json格式字符串


**输入参数（新增）：**

+------------------------+------------+------------+------------------------------------------------+
|**参数**                |**数据类型**|**是否必须**|**说明**                                        |
+------------------------+------------+------------+------------------------------------------------+
| username               | string     | 是         | 用户名                                         |
+------------------------+------------+------------+------------------------------------------------+
| password               | string     | 是         | 用户密码                                       |
+------------------------+------------+------------+------------------------------------------------+


**输出参数：**

+------------------------+------------+------------+------------------------------------------------+
|**参数**                |**数据类型**|**是否必须**|**说明**                                        |
+------------------------+------------+------------+------------------------------------------------+
| token                  | string     | 是         | 用户凭证                                       |
+------------------------+------------+------------+------------------------------------------------+

**登录请求示例：**
::

    {
		"username":"admin",
		"password":"admin"
	}


**POST登录返回数据示例：**
::

    {
        "token": "faisjdjasidjijas23d98c3fhhcsd8"
    }


**POST登录错误返回示例：**
::

    {
		"detail":"The username or password is not correct"
	}
