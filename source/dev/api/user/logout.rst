登出
======================

说明
-----------------------------------------------------------------------------------------------------------------------
- 用户主动登出系统，并失效token


**请求方式：**   POST

**请求地址：**   /api/logout/


**Content-Type：**
::

    application/json —— 表示传入数据为json格式字符串

**输入参数：**

+------------------------+------------+------------+------------------------------------------------+
|**参数**                |**数据类型**|**是否必须**|**说明**                                        |
+------------------------+------------+------------+------------------------------------------------+
| token                  | string     | 是         | 用户名                                         |
+------------------------+------------+------------+------------------------------------------------+


**输出参数：**

+------------------------+------------+------------+------------------------------------------------+
|**参数**                |**数据类型**|**是否必须**|**说明**                                        |
+------------------------+------------+------------+------------------------------------------------+
|                        | string     | 是         |                                                |
+------------------------+------------+------------+------------------------------------------------+

**登出请求示例：**
::

    {
        "token":"WmZPC08FASHCuo9Ic9edP66bFLwb3ChqVJbBUN1iYEEa8PyCeyWrM3M6SenwzVYM"
    }


**POST登出返回数据示例：**
::

    {

    }


**POST登出错误返回示例：**
::

    {
        "detail":"The username or password is not correct"
    }
